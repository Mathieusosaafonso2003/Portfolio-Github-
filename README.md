# Calculadora Simple en Java

Este proyecto es una calculadora básica desarrollada en Java. Permite realizar operaciones aritméticas simples como suma, resta, multiplicación y división.

## Requisitos

- Java 8 o superior
- IDE como IntelliJ IDEA, Eclipse, o cualquier otro editor compatible con Java.
