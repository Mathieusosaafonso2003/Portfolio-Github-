# Proyecto Calculadora

## Descripción

Este proyecto es una calculadora simple desarrollada con HTML, CSS y JavaScript. Permite realizar operaciones matemáticas básicas como suma, resta, multiplicación y división.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Archivos del proyecto

- `Index.html`: Contiene la estructura principal de la calculadora.
- `styles.css`: Archivo de estilos para la interfaz de usuario.
- `script.js`: Maneja la lógica de la calculadora.
- `Calculator.ttf`: Fuente personalizada utilizada en el diseño.

## Cómo ejecutar el proyecto

1. Descarga o clona el repositorio.
2. Abre el archivo `Index.html` en cualquier navegador web.

## Funcionalidades

- Interfaz sencilla y fácil de usar.
- Operaciones matemáticas básicas: suma, resta, multiplicación y división.
- Diseño responsivo con CSS.
